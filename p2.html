Project: IAT235 Information Website Design

Project Description

This is a project required to design a information website for a fictional moutain safety and rescue team. As a non-profit association, this website mainly accounts for popularizing the safety guidelines and rescue knowledge about two topics: Avalanche and Hypothemia, as well as intorudcing the team itself. In this case, the design focus is to accurately express the correct guideline information to the users while the layout of the website maintains the aesthetics level, and the former should be the most important part. This website was first designed in Figma for layout tests, and then to create out the frameworks step-by-step as my coding skills were just in a primary level at that stage. After many polishes and breakdowns finally the pages were having some consistent layouts and the necessary elements were included. With the filling of the safety knowledge the website shall be ready to be used for advertising purposes.

Reflection

From my perspective, I am satisfied what we (this project is done with two partners) have created out. IAT 235 was the first time that I attaced website designing. At that stage I was researching and doing some pratical designs in order to try my best to make the website look professional. Basically the page layouts are the same for the pages which account for same purposes. The color scheme for the website were also selected to fit the topics well. The usability tests were not bad as most of the functions worked well. The information sections combined both text and visualize elements to create comfort reading experiences for users. What I feel not perfect is the sizes of the elements. As this is the first approach, some of the size decisions seemed to be inappropriate as they may look too big on screens. The compositions of some of the elements can also have some improvements. But overall I think in this project we successfully expressed the information as planned and the whole website looks good in terms of color harmonic and theme relevance. I have gained some knowledge about informative designs and have learned many basic rules in website development.

Softwares/Skills used:

HTML5 + CSS3 + Adobe Photoshop + Figma
